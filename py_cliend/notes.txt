1. HttpResponse

HttpResponse is a general-purpose response. It can send almost any content to the browser.

It supports:
------------
✅ Strings
✅ HTML
✅ Bytes
✅ Files (with related response classes like FileResponse)
✅ Any object that can be converted to a string
---------------------------------------------------------

JSON supports:
--------------
✅ Dictionary (dict)
✅ List (list) (use safe=False for a top-level list)
✅ String (str)
✅ Integer (int)
✅ Float (float)
✅ Boolean (True / False)
✅ None (becomes null in JSON)
----------------------------------------------------
